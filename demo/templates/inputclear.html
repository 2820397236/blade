inputclear组件

基本参数说明


    @param template      {String}    自定义模板
    @param clearCallback {Function}  点击删除的callback
    @param getInputVal   {Function}  获取当前input的值
    @param datamodel    {        数据模型
        parent          {String}    父级Dom的class或id,委托绑定
        container       {String}    input的class或id
        clearClass      {String}    删除按钮的class
        offset          {Object}    按钮的位置可设置top,left,bottom,right
        clearEventClass {String}    默认删除按钮事件class，一般情况不作赋值
    }
